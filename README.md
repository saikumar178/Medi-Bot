# Medi-Bot

### How to run 
- clone the repo
- create a venv
    >python -m venv venv
- activate the venv 
    >venv/Scripts/Activate
- install requirements
    >pip install -r requirements.txt
- run the main app
    >uvicorn main:app --reload --port 8000

### Note
- make sure to keep GROQ_API_KEY in .env file
- to deactivate venv
    >Deactivate
